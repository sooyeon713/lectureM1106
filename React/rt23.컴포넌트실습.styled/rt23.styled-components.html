컴포넌트 스타일링 기법을 학습한다. 컴포넌트의 작성 원칙은 수정없이 재사용 가능해야 한다. 컴포넌트의 html, style, javascript를 한 곳에 저장하는 것을 권장한다. React의 스타일링 방식 1. inline css 방식 JS 표기법을 사용하여 컴포넌트 안에서 style을 만드는 방법 2. import css 방식(★) import './파일명.css'; 3. styled-components 모듈(★★) CSS-in-JS 방식 tagged-template-literal 을 이용 import styled, { css } from 'styled-components'; 4. module css 방식 import 모듈명 from './파일명.modules.css'; CSS 표기법을 사용하는 스타일링할 때 사용되는 라이브러리가 styled-components 이다. styled-components 는 CSS 표기법을 사용하여 인라인 스타일링을 만들 수 있게 해주는 라이브러리 이다. styled-components를 사용하여 리액트 컴포넌트를 생성하기 위해서는 styled.[HTML 태그] 형식과 템플릿 리터럴(`)을 사용한다. 예시 1) >>> styled-components 적용 전 <style> .App { text-align: center; } </style> <div className="App"> ...중략 </div> >>> styled-components 적용 후 const StyledContainerDiv = styled.div` text-align: center; `; <StyledContainerDiv> ...중략 </StyledContainerDiv> React CRA(create-react-app) 을 사용하여 styleprj01 프로젝트를 만들고 styled-components 컴포넌트에 적용하여 본다. * CRA 는 create react app 의 약자. ################################## 1. 작업 개요 ################################## ################################## 2. 프로젝트 생성 작업 소요시간: 20분 ################################## step 1-01. node / npm 버전 업그레이드 $ node -v # node 버전 확인 v16.16.0 $ npm -v # npm 버전 확인 8.18.0 $ npx -v # npx(=npm extension) 버전 확인 8.18.0 $ npm install -g npm@latest # npm 최신 버전 설치 $ npm install -g yarn@latest # yarn 설치. yarn은 페이스북에서 만든 npm. $ yarn -v # yarn 버전 확인 1.22.19 step 1-02. 프로젝트 생성 $ npx create-react-app styleprj01 # 프로젝트명은 소문자로 만들어야 함.
